# Vector Calculator
Just a useless calculator.  

## Features
Those who know.  

## Build VSIX
```bash
npm install
npm run compile
npx vsce package
```

## Install VSIX
```bash
code --install-extension <path/to/vsix>
```

## Usage
Those who know.  
- Ctrl+Alt+E
- Ctrl+Alt+M

### Settings
`vectorCalculator.modelDataPath` (global folder, scanned recursively)
- Default: `${userHome}/vector-calculator/model-data`
